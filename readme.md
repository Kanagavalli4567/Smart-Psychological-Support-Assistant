🧠 MindMate AI – Your Intelligent Mental Wellness Companion

Empowering mental well-being through meaningful conversations, emotional support, and AI-driven assistance.

🌟 About the Project

MindMate AI is an intelligent mental health support chatbot developed to provide users with a safe, accessible, and supportive environment for expressing their thoughts and emotions. The chatbot engages users in interactive conversations, offers emotional guidance, and promotes mental wellness through AI-powered responses.

Built with HTML, CSS, JavaScript, Python, and SQLite, MindMate AI combines a clean user interface with a robust backend to create a seamless and engaging user experience.

🚀 How It Works
User enters a message through the chat interface.
The message is processed by the Python backend.
The chatbot analyzes user input and generates a relevant response.
Conversations are stored securely in SQLite.
The response is displayed instantly in the chat window.

⚙️ Installation Guide
Clone the Repository
git clone https://github.com/your-username/MindMate-AI.git
cd MindMate-AI
Create Virtual Environment
python -m venv venv
Activate Environment

Windows

venv\Scripts\activate

Mac/Linux

source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
Run the Application
python app.py

Open your browser and visit:

http://127.0.0.1:5000
